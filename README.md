# SplitManager
Sistema software per la gestione delle spese condivise all'interno di gruppi di persone.

## Funzionalità
- Gestione gruppi e membri
- Tracciamento spese condivise
- Calcolo automatico saldi
- Ottimizzazione rimborsi
- Sistema di notifiche

## Architettura
Il progetto segue un'architettura multilayered:
- **Presentation Layer**: Interfaccia a riga di comando (CLI)
- **Business Logic Layer**: Gestione della logica applicativa
- **Domain Model Layer**: Entità del dominio
- **Persistence Layer**: DAO con PostgreSQL


## Tecnologie Utilizzate
- **Linguaggio**: Java 21
- **Database**: PostgreSQL
- **Database Connectivity**: JDBC
- **Testing**: JUnit 5
- **Diagrammi**: Luchidchart
- **IDE**: IntelliJ IDEA
